### Node.js backwards imcompatibility example
This example works in nodejs 12.15.0, but doesn't work in nodejs 12.16.0.
To run it, use following commands:
```
$ git clone https://github.com/SagePtr/test-issue452
$ cd test-issue452
$ npm ci
$ npm run webpack
```
